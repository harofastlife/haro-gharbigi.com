Code repositories and technical projects demonstrating development involvement.

GitHub houses repositories that showcase technical involvement in software development, from personal projects to collaborative work advancing healthcare technology solutions.

While my primary role centers on business leadership and strategy, maintaining a presence on GitHub reflects the importance of understanding the technical foundations of the products we build at AxiaASC. This hands-on engagement with code ensures informed decision-making when guiding development teams.

The repositories span various projects related to healthcare technology, demonstrating practical applications of programming concepts in service of medical facility operations. Open-source participation aligns with a broader philosophy of knowledge sharing that extends across my professional activities. Just as I share insights through writing and presentations, GitHub provides a platform for contributing to the technical community that drives innovation in our sector.

For fellow technologists and developers interested in healthcare AI applications, these repositories offer glimpses into the kinds of problems we're solving and the approaches we're taking. The code tells a story of continuous learning and adaptation in a rapidly evolving field.
