# haro-gharbigi.com

Personal website for Haro Gharbigi - Healthcare Technology Innovator & AI Strategist.

## About

This is the official website for Haro Gharbigi, founder of AxiaASC, showcasing expertise in healthcare technology, artificial intelligence, and business strategy.

## Features

- Modern, responsive design with navy/blue theme
- SEO-optimized with structured data (JSON-LD)
- Open Graph and Twitter Card meta tags
- Smooth scrolling navigation
- Mobile-first responsive layout

## Sections

- **Hero** - Introduction and call-to-action
- **About** - Background and experience overview
- **Ventures** - Featured business ventures (AxiaASC, Smash.com, Rejuvie Med Spa)
- **Journey** - Timeline of professional experience
- **Expertise** - Areas of specialization
- **Insights** - Links to published content and thought leadership
- **Connect** - Social media and professional profile links

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript (smooth scrolling)
- Google Fonts (Playfair Display, Inter)
